#Jokenpô Arena

Um jogo clássico de Pedra, Papel e Tesoura desenvolvido com HTML, CSS e JavaScript puro.

O projeto foi criado com o objetivo de praticar manipulação do DOM, eventos e lógica de programação no desenvolvimento web.

---

#Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

#Funcionalidades

- Escolha entre Pedra, Papel ou Tesoura
- Geração aleatória da jogada do computador
- Sistema de pontuação
- Atualização dinâmica do placar
- Layout responsivo

---

#Aprendizados

Durante o desenvolvimento deste projeto, foram praticados:

- Manipulação do DOM
- Uso de querySelector
- addEventListener
- Estruturas condicionais
- Organização de código em arquivos separados
- Responsividade com CSS

---

#Objetivo do Projeto

Este projeto foi desenvolvido como prática acadêmica para consolidar conhecimentos fundamentais de JavaScript e desenvolvimento web.
